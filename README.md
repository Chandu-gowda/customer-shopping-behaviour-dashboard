# customer-shopping-behaviour-dashboard
## Analysis Summary

The Customer Shopping Behaviour Dashboard provides insights into customer purchasing patterns, spending habits, and product preferences. The dashboard was developed using Microsoft Excel to analyze customer behavior and support business decision-making.

### Key Findings

* Identified the most frequently purchased product categories.
* Analyzed customer spending patterns and purchase frequency.
* Compared purchasing behavior across different customer segments.
* Highlighted trends in customer preferences and buying habits.
* Identified opportunities for targeted marketing and customer retention strategies.

### Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

### Business Value

The dashboard helps businesses understand customer behavior, improve marketing strategies, and make informed decisions based on purchasing trends and consumer preferences.
